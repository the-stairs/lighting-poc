### 편집모드(가칭)

- 현재까지 개발 완료된 모든 상태를 포함하여 편집모드로 명명
- 각 오브젝트의 형태, 투명도, Position등 속성을 정의하는 기능 수행
- 편집모드/촬영모드 전환 기능 추가
  - 예시
    - 편집모드: F3
    - 촬영모드: F4
- 각 오브젝트가 투사되는 시작 시간을 부여
  - 기본값: 0.0초
  - 개별 object가 별도의 시작시간을 가질 수 있도록 속성 부여
- 각 오브젝트의 지속 시간 부여 기능 추가
  - Textfield를 통한 사용자 입력/프리셋 저장
    - 사용자입력단위: 0.0초
  - 설정되지 않은 Object는 기본값 설정
    - 기본값은 사전 정의된 전역값에 의해 부여하고, 사용자가 변경할 수 있는 Textfield를 UI에 반영
    - 개발시 Default는 1.0초로 지정
  - 향후 Object들의 Timer Group이 구현될 수 있음을 고려하여 개발 필요

### 촬영모드(가칭)

- 촬영모드 전환시 화면에 표시되는 상태는 검은 화면(모든 Object의 Visible 속성 False)
- 촬영 트리거 동작시 Object의 Timer 활성화
  - 트리거 핫키: F5(예시)
- 각 Object는 다음 기능을 수행
  - 점등: 트리거 발생시점으로부터 속성에 부여된 시작시간이 지나면 Visible=True
  - 소등: 점등 시점으로부터 속성에 부여된 지속시간이 지나면 Visible=False
